# Need Chameleon? (Windows only—since it was always just a Windows product)

If you need a Chameleon License Code, you can install **Iguana 6.1.5** or **Iguana 6.2.0** on Windows, as both versions include the latest version of **Chameleon**.  See [downloads](download.md).

“Latest,” as in the latest Elvis records, since that’s about how old Chameleon is—lol.

When you get an [Iguana license, it also licenses Chameleon](../license.md), so this is probably the smartest way in 2026 to obtain a piece of software that really should have been retired at least a decade ago.

And yes, you still need to license it. Eliot Muir is still alive and still needs to buy food, pay for living costs, and other stuff. He's just an accommodating guy who really believes in supporting his customers—even if some of them can be a little mean.

The licensing system will be refined quickly over time.
