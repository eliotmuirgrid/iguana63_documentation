# What does the maintenance expiry date mean on license codes?

This date affects whether or not your license code will work on an upgraded version
of Iguana.

It dates to when there were more regular releases of Iguana Classic and it probably
should be improved on when we update the licensing system.

With Iguana 6.3 the system will be replaced.  It's long over due for replacement.

So in general clients shouldn't worry about the maintenance date.

New license codes will be issued with Iguana 6.3.
