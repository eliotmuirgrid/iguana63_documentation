# Editing the source code of vmd files in JSON format is often more efficient 


Editing the source code of VMD files in JSON format is often more efficient.

By right-clicking in the project editor, you can edit these files and view their source, which is in a deliberately human-readable JSON format.

This is often the easiest way to create VMD files quickly, as you can copy and paste fields, making it simple to create multiple files at once.

This approach can be quite handy when dealing with the problem of [Z segments](zsegment.md).
